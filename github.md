# GitHub VSCode

## 1. Prerequisites

Git Installed: Ensure Git is installed on your system. Check by running:

$ git --version

If not installed, download Git.

<code>
$ scoop install git
</code>

<code>
$ git config --global user.name "John Doe"
$ git config --global user.email johndoe@example.com
</code>
